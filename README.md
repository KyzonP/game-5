Game 5 of the 20 Games Challenge. A Pac-Man remake made using Godot!

Project is currently playable as a web game on my Itch.io page: https://morpeth-jetsan.itch.io/pac-sam

In total the project took 1338 minutes (About 22 hours). This was across 8 seperate days (Average 167 minutes a day, just under 3 hours).
